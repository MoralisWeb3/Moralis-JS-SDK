AptosAddressInput​

Ƭ AptosAddressInput: AptosAddress | string

Valid input for a new AptosAddress instance. This can be an existing AptosAddress or a valid address string.

Example

"0x54ad3d30af77b60d939ae356e6606de9a4da67583f02b962d2d3f2e481484e90"

Example

AptosAddress.create("0x54ad3d30af77b60d939ae356e6606de9a4da67583f02b962d2d3f2e481484e90")
